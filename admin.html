📚 Quản lý sách & giáo trình
💡 Quản lý linh hoạt
Thêm sách mới, chỉnh sửa tên/màu sắc/icon, tắt/bật hiển thị cho học sinh, sắp xếp thứ tự hiển thị.
Mỗi sách có thể có nhiều Level / Lớp — học sinh sẽ chọn sách → chọn level → chọn unit để luyện nói.
✅ Tất cả sách được quản lý từ đây
Thêm, sửa, xóa, ẩn/hiện bất kỳ sách nào. Lần đầu hãy chạy file insert_default_books.sql để thêm sách mặc định vào hệ thống.
📭
Chưa có sách tuỳ chỉnh. Nhấn "+ Thêm sách mới" để bắt đầu!
📝 Quản lý nội dung giáo trình
💡 Cách dùng nhanh nhất
Lần đầu: Paste toàn bộ nội dung 1 quyển sách → AI tự chia từng unit → Lưu hết 1 lần 🚀
Cập nhật: Nhấn ✏️ Sửa ở unit muốn thay → Paste nội dung mới → Lưu
🤖 Paste cả quyển sách — AI tự chia theo unit
Paste toàn bộ nội dung từ NotebookLM (hoặc bất kỳ nguồn nào). AI sẽ tự nhận diện từng unit, trích xuất từ vựng + mẫu câu + gợi ý luyện nói rồi lưu tất cả vào hệ thống một lần.
✅ Cách 1: Import JSON (Khuyên dùng)
Paste văn bản lên Claude.ai → Claude tạo file JSON → import vào đây. Không cần API key!
🔑 Cách 2: AI tự động (Cần API key)
Cần Anthropic API key trả phí. Nếu có key, nhập vào đây để dùng.
